AAFAnalyzer: Simple AAF Report Manager
    (ReadMe / Help)

Description:
  To help you create reports from a set of AAF files.
    - a requirements report in .HTML format
    - a summary report in .TXT format

Requirements:
  Python 2.5 installed:
    See http://www.python.org/ftp/python/2.5/python-2.5.msi
  You should know the location of the AAF file or files.

Instructions:
  Choose a disk by typing the letter at the disk prompt.
  Navigate to the folder (left panel) that contains your AAF file(s).
  Select the AAF file and press the Report button.

  When the report file viewer (right panel) refreshes, select one of the files and press the View button.
     To view AAF files you will need Salto installed in the default location (see http://www.diskstream.com/support.html ).

Known Limitations:
  a) Available only for Windows XP.
  b) Building of a single report from multiple AAF files not supported.
  c) Crashes on the opening of a "\System Volume Information" folder.
